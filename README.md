The Image Color Palette Extractor is a Python application that allows users to extract dominant colors from an image and generate color palettes. 
The application provides a user-friendly interface to open images, auto-generate color combinations, and search for predefined color schemes.

Extract Dominant Colors: Extracts the dominant colors from an image using the ColorThief library.
Auto-Generate Color Combinations: Generates random color combinations.
Search Color Scheme: Allows users to search for predefined color schemes by description.
Copy to Clipboard: Users can copy the hex code of any color to the clipboard.
Scrollable UI: The color palette display area is scrollable, allowing users to view all colors easily.
